# Project Title: Programming Data Structures and Algorithms Game System

## 📌 Description

This project is a collection of five interactive algorithm-based games developed as part of the **Programming Data Structures and Algorithms (PDSA)** module. The system is designed using a full-stack architecture with a **Spring Boot backend** and a **React frontend**.

Each game demonstrates the application of fundamental data structures and algorithms such as graph traversal, backtracking, optimization techniques, and multithreading. The system not only provides an interactive gaming experience but also evaluates algorithm efficiency by recording execution time and comparing multiple approaches.

---

## 🎯 Objectives

* Apply data structures and algorithms to real-world problems
* Compare multiple algorithmic approaches for each problem
* Measure and analyze execution time
* Provide an interactive user interface for gameplay
* Store and manage results using a database

---

## 🕹️ Games Included

### 1. Minimum Cost Assignment Game

A simulation where tasks are assigned to employees with the goal of minimizing total cost.

* Algorithms Used: Hungarian Algorithm, Greedy Approach
* Features: Random cost generation, performance comparison

---

### 2. Snake and Ladder Game Problem

A board-based game where players determine the minimum number of dice throws required to reach the final cell.

* Algorithms Used: Breadth-First Search (BFS), Dynamic Programming
* Features: Random board generation with snakes and ladders

---

### 3. Traffic Simulation Game

A network flow problem where players calculate the maximum flow from source to sink.

* Algorithms Used: Ford-Fulkerson, Edmonds-Karp
* Features: Graph visualization, capacity randomization

---

### 4. Knight’s Tour Game

A chess-based problem where a knight must visit every square exactly once.

* Algorithms Used: Backtracking, Warnsdorff’s Algorithm
* Features: Supports 8x8 and 16x16 boards

---

### 5. Sixteen Queens Puzzle Game

An advanced version of the queens puzzle on a 16x16 board.

* Approaches Used: Sequential Algorithm, Multithreaded Algorithm
* Features: Solution tracking, duplicate prevention, performance comparison

## ⚙️ Installation Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/pdsagame-project
```

### 2. Navigate to Project Directory

```bash
cd pdsagame-project
```

---

### 3. Backend Setup (Spring Boot)

```bash
cd backend
```

* Configure `application.properties` with your MySQL credentials
* Run the application using your IDE (IntelliJ / Eclipse)

---

### 4. Frontend Setup (React)

```bash
cd frontend
npm install
npm run dev
```

## ▶️ Usage

* Launch the application and access the main game menu
* Select a game from the list
* Provide required inputs (board size, answers, etc.)
* Submit your answer and view results (Win/Lose/Draw)
* The system will:

  * Calculate correct answers using algorithms
  * Compare execution times
  * Store results in the database
